===========================================
VR 오큘러스 눈 상태 관제 시스템 사용 설명서
===========================================

본 프로그램은 오큘러스 기기와 PC를 연결하여 실시간으로 시표를 조작하고 검제를 수행하는 의료용 관제 시스템입니다. 설치할 필요 없이 본 폴더만 있으면 어디서든 바로 실행이 가능합니다.

-------------------------------------------
1. 준비하기 (가장 중요)
-------------------------------------------
 1. 먼저 오큘러스 기기의 전원을 켜주세요.
 2. PC와 오큘러스 기기를 유선 통신 케이블(USB C타입 등)로 단단히 연결해 주세요.
 3. (선택) 만약 오큘러스 화면 내에서 "데이터 접근 허용" 이나 "USB 디버깅 허용" 이라는 팝업이 뜬다면 반드시 [허용(항상 허용)]을 눌러주셔야 합니다.

-------------------------------------------
2. 실행하기
-------------------------------------------
준비가 끝났다면, 본 폴더 안에 있는 [ start_test.bat ] 파일을 더블클릭해서 실행하세요.

* 실행 시 일어나는 일들:
 1. 검정색 터미널 창(콘솔 창)이 열리며 오큘러스와의 연결 터널을 자동으로 뚫습니다.
 2. 이어서 "Eye_Server.exe" 가 켜지며 백그라운드에서 동작을 시작합니다.
 3. 몇 초 뒤, 크롬 등 웹 브라우저가 스르륵 열리며 "VR 검사 실시간 관제 화면"이 나타납니다.

★ 주의: 처음에 나오는 검은색 콘솔 창을 실수로 X 버튼을 눌러 끄면 시스템 작동이 일제히 멈춥니다! 
최소화해서 바탕화면 밑으로 내려놓거나 그대로 띄워두고 쓰시면 됩니다.

-------------------------------------------
3. 맵핑 오류나 딜레이가 있는 경우
-------------------------------------------
* "서버는 켜졌는데 웹 화면에 오큘러스 연결(VR 연결)이 뜨지 않아요!"
  -> 해결책: USB 선을 뺐다 다시 꽂은 뒤, [ start_test.bat ] 파일을 다시 한번 실행해 주세요.

* "웹에서 버튼을 누르면 기기 화면이 잘 바뀝니다!"
  -> 정상입니다. 의사 선생님의 웹 모니터 화면에서 마음껏 조명 크기와 밝기를 통제하며 검사를 진행하시면 됩니다.

-------------------------------------------
4. 프로그램 종료
-------------------------------------------
검사가 모두 끝나셨나요? 
 1. 띄워두셨던 [ 웹 브라우저 화면 ]을 닫아주세요.
 2. 밑으로 내려놨던 [ 검정색 터미널 창 ]을 X 버튼을 눌러 닫아주시면 서버와 통신은 완전히 꺼집니다. 
 3. 오큘러스 기기의 선을 조심히 빼면 마무리 지어집니다.

수고하셨습니다!
