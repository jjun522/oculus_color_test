# VR 시신경병증 검사 가이드라인 (Oculus 3S)

본 문서는 메타 퀘스트 3S(Meta Quest 3S)와 실시간 웹 관제 시스템을 활용하여 환자의 시신경 이상 여부(특히 RAPD 및 특정 시야 결손)를 검사하는 절차를 안내합니다.

## 1. 장비 세팅 및 연결

1. **프로그램 실행:** `Eye_Test_Hospital_Deploy` 폴더 내의 `Eye_Server.exe` (또는 `start_test.bat`)를 더블클릭하여 서버를 실행합니다.
2. **웹 관제 패널 접속:** PC의 웹 브라우저(Chrome 권장)에서 `http://localhost:12346` 에 접속합니다.
3. **VR 기기 착용 및 연결:** 환자에게 오큘러스 3S를 착용시킵니다. 기기 내의 검사용 앱을 실행하면 자동으로 PC 서버와 연결됩니다.
   - 웹 패널 우측 상단에 `🟢 VR: (연결된 ID)`가 뜨면 정상입니다.
   - 십자가(마커)가 화면 끝에서 끝까지 이어지는 4분할 형태로 나타나는지 확인합니다.

## 2. 검사 전 피험자 안내 (매우 중요)

- **자세:** 의자에 편안하게 기대어 앉도록 합니다.
- **시선 고정:** "화면 전체에 하얀색 큰 십자가 선이 보일 겁니다. **검사가 끝날 때까지 무조건 십자가가 교차하는 정중앙만 똑바로 쳐다보세요.**" 라고 지시합니다.
  - *주의: 퀘스트 3S는 하드웨어 센서 한계로 기기 스스로 환자의 눈동자 위치를 추적(아이트래킹)할 수 없습니다. 따라서 환자가 스스로 중앙을 잘 응시하도록 사전에 단호하게 교육하는 것이 가장 중요합니다.*

## 3. 검사 진행 순서

웹 관제 패널의 `🎮 리모컨` 탭을 이용해 검사를 진행합니다.

### A. RAPD (상대애퍼런트동공결손) 검사 (기본 모드)
1. **모드 설정:** `[양안 모드 (RAPD 준비)]` 버튼을 클릭합니다.
2. **밝기 사전 설정:** `[좌 30 / 우 80]` 또는 `[좌 80 / 우 30]` 버튼을 눌러 양안의 밝기 단차를 줍니다.
3. **플립 시작:** `[시작]` 버튼을 누르면 1.5초(기본값) 간격으로 좌안과 우안의 화면이 교대로 깜빡입니다.
   - 이 때 의료진은 환자의 동공 반응을 관찰하여 RAPD 양성/음성 여부를 판별합니다.

### B. 4분할 시야 밝기 검사
1. **모드 변경:** `[좌안 4분할]` 또는 `[우안 4분할]` 버튼을 클릭합니다.
2. **검사 방식:**
   - Q0(우상), Q1(우하), Q2(좌상), Q3(좌하)의 4개 구역에 대해 독립적으로 밝기를 조절합니다.
   - 우측 패널의 `독립 수치 제어`에서 특정 분면의 밝기 수치를 낮추거나 높인 뒤 `Set`을 누릅니다.
   - 환자에게 "어느 쪽이 더 어둡게/밝게 보입니까?" 라고 질문하여 시야 결손 부위를 파악합니다.

### C. 2분할 (코쪽/귀쪽) 검사
1. **모드 변경:** `[좌안 2분할]` 또는 `[우안 2분할]` 버튼을 클릭합니다.
2. **검사 방식:**
   - 코쪽(Nasal)과 귀쪽(Temporal)의 밝기를 독립적으로 제어합니다.
   - 반맹 등의 증상을 확인할 때 사용합니다.

### D. 원 모드 (256 Grid) 정밀 검사
1. **설정 변경:** `[설정]` 탭으로 이동하여 '타겟 형태'를 **256 Grid (원)** 으로 변경합니다. (기본 크기: 8x8)
2. 전체 화면이 아닌 작은 원들의 배열로 화면이 바뀝니다. 시야의 특정 암점을 더 세밀하게 찾고자 할 때 활용합니다.

## 4. 검사 종료 및 데이터 기록

- 웹 화면 중앙 미러 스크린에 표시되는 Lux 수치와 밝기(%) 로그를 확인합니다.
- 검사가 끝나면 `Eye_Server.exe`를 종료하고 VR 기기를 벗게 합니다.
